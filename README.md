## Project Overview
This project was developed as a technical assessment for a Full-Stack practicum, completed within a **strict 2-hour timeframe**.

## My Contribution & Workflow
While I utilized AI tools to accelerate the coding process, all core architectural decisions and system design were independently executed by me, including:

* **System Architecture:** Planning the folder structure and project organization (Layered Architecture).
* **Database Design:** Defining the Schema, table relationships, and ERD logic.
* **Data Persistence:** Manually managing and executing **EF Core Migrations**.
* **AI Orchestration:** Guiding the AI through precise prompting to implement the Business Logic based on my architectural requirements.
